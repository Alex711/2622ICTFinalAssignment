Design Document for DMS Travel

The following details the specific techniques that were used to create the design for DMS Travel.

Repetition

Repetition has been used throughout the entire website to tie each page of the website together, to create a more professional look and to overall create a nice and clean "feel" to the website.

Evidence of Repetition can be found in the navigation bar, colour schemes and button/link colour schemes

Pattern

To keep the image grid layout design in a certain format; Each grid is the same size and has the same background colour.

Emphasis / Visual Hierarchy

Emphasis has been an important principle in the design of this website; As there can be many images, it is important to give emphasis to the images which are most important at the time (eg Liking / Viewing a photo).

For example within the image display (grid layout) when a user has their mouse hovered on an image, the image automatically zooms in as it is what the user is focusing on (most important at the time).

Another example is placement of buttons; The splashscreen page has the button placement closer to the centre and larger compared to other buttons to create a visual hierarchy and emphasise that this is an important aspect of the website.
